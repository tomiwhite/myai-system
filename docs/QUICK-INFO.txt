│ 6️⃣  CO JEŚLI COŚ NIE DZIAŁA?                                           │
└──────────────────────────────────────────────────────────────────────────┘

❌ "VS Code nie znaleziony"
   → Zainstaluj: https://code.visualstudio.com/
   → Uruchom ponownie ikonę

❌ "Python nie znaleziony"
   → Zainstaluj Python 3.13+: https://www.python.org/downloads/
   → WAŻNE: Zaznacz "Add Python to PATH"
   → Uruchom ponownie ikonę

❌ "Google Drive .env not found"
   → Włącz Google Drive
   → Sprawdź: G:\Mój dysk\podreczne - mysite\myoffice 4-0 dokumentation\.env

⚠️ Inne problemy?
   → Przeczytaj komunikaty w oknie PowerShell (często podają rozwiązanie)
   → Zapytaj AI Asystenta w VS Code: "System nie działa, co robić?"
